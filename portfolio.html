What Happened to the Dinos?

Scratch

A project assignment early in the CS50 course, where students are free to choose any type of project but subject to requirements such as using at least two sprites, three scripts, one condition, one loop, one variable and one sound. VicTrees Vs. Deforestation

React | Node/ExpressJS | ThreeJS | Bootstrap | Leaftlet | Global Forest Watch | Instagram | New Relic | Heroku

A collaborative web application project to create awareness and actionable steps towards combating deforestation. Our team entered Hack the Planet Contest with 'VicTrees fight Deforestation Victoriously' and won runner-up in the New Relic Hackathon!

MySchoolAdmin - Grading MVP

Vue | Bootstrap | ApexCharts | Node/ExpressJS | MySQL

A minimum viable product(MVP) web application for teachers and school administrators to compile grades for student report cards. This is a work in progress with an extensive feature wish-list plan. I am using this application personally for school administration work, at a private homeschool centre where I am a part-time volunteer.
